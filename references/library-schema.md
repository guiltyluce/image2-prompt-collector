# Image2 Prompt Gallery Schema

## Main Table

Configure the target Feishu Base and table through environment variables or CLI
arguments:

- `FEISHU_BASE_TOKEN` / `--base-token`
- `FEISHU_TABLE_ID` / `--table-id`
- optional `--attachment-field` (default: `例图`)

Fields:

| Field | Type | Required | Notes |
|---|---|---:|---|
| `资产ID` | text | yes | Generated by importer, e.g. `PG-20260428-001`. |
| `标题` | text | yes | Short title; no source spam. |
| `例图` | attachment | yes | The visible example image. Main table records must not be created without this. |
| `提示词` | text | yes | Complete prompt, directly usable for image generation. |
| `项目标签` | single select | yes | Use `public-curation` for newly curated public-source entries unless updating a project-specific series. Existing values include `gpt_image_2_skill`, `awesome-gpt-image-2`. |
| `类型标签` | single select | yes | Prefer existing values `Prompt` or `案例Prompt`; use `案例Prompt` for public examples. |
| `风格标签` | single select | yes | Choose an existing category when possible. Use `其他应用场景` if no good fit. |
| `来源平台` | single select | yes | Existing values: `官方/项目整理`, `小红书`, `X/Twitter`, `Reddit`, `未提供`, `其他`. |
| `来源` | text | recommended | Creator handle, post title, repo, or page title. |
| `来源链接` | text | recommended | Public URL. |
| `尺寸或比例` | text | optional | e.g. `9:16`, `1:1`, `1536x1024`. |
| `资源状态` | single select | yes | `可直接复用`, `图片需复核`, or `缺例图待生成`. Main-gallery imports should normally be `可直接复用`. |
| `精选` | checkbox | optional | True only for editorial picks. |
| `用途说明` | text | recommended | One-sentence reuse guidance. |
| `备注` | text | optional | Import batch, moderation notes, or caveats. |

## Material Pool

If you keep a separate material-pool table, configure it in your own workspace.
The importer in this repository writes only to the main prompt table.

Use only for images that do not yet belong to a usable prompt record. Do not duplicate every main-gallery image here.

## Candidate JSON Schema

`scripts/import_candidates.py` accepts a JSON array:

```json
[
  {
    "title": "Apple-style natural science poster",
    "prompt": "Generate a 9:16 premium natural science poster...",
    "image_path": "./prompt-gallery-intake/media/example.png",
    "source_platform": "X/Twitter",
    "source": "@creator or post title",
    "source_url": "https://example.com/post",
    "style_tag": "海报与排版",
    "type_tag": "案例Prompt",
    "project_tag": "public-curation",
    "size_or_ratio": "9:16",
    "featured": false,
    "usage_note": "适合替换物种名与四个科普栏目后批量生成系列海报。",
    "notes": "Collected from public post."
  }
]
```

Required: `title`, `prompt`, `image_path`. The importer skips records without a local image.

## Accepted Tags

If a source produces a new label, map it to one of these before import unless the user asks to extend the Base schema:

- `UI与界面`
- `图表与信息可视化`
- `海报与排版`
- `商品与电商`
- `品牌与标志`
- `人物与角色`
- `建筑与空间`
- `摄影与写实`
- `插画与艺术`
- `场景与叙事`
- `其他应用场景`
- Existing project-derived categories already in the table can also be reused.
